# Wallpaper Pattern Generator

A browser-based pattern generator that creates unique, procedurally-generated wallpapers and backgrounds. Explore different algorithmic designs and export custom patterns for your desktop, mobile device, or creative projects.

## Features

- Multiple pattern generation algorithms (8+ unique generators)
- Procedurally-generated designs for endless variations
- Interactive web interface
- Client-side generation - no server required
- Instant preview and customization
- Exportable high-resolution patterns

## Usage

Open any of the HTML files in your web browser to access different pattern generators:
- `Wallpaper Pattern Generator 0.html` through `Wallpaper Pattern Generator 7.html`

Each generator produces a distinct style of pattern using different algorithms and visual approaches.

## Technical Details

Built with vanilla HTML, CSS, and JavaScript. Uses HTML5 Canvas API for rendering procedural patterns. All generation happens client-side for privacy and performance.
